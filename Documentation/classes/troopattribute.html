Variant calc_move_cost(strength: float, from: Vector2i, to: Vector2i, board: Board)

Virtual function which overrides movement costs. Overrides to this function can return a non-null value to change the move calculation. Otherwise, returning null uses the default tropp calculation. For information on the parameters, see Troop._calc_move_cost
